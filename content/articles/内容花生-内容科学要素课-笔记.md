# 内容花生 - 内容科学要素课（精炼笔记）

> 来源：内容花生付费课程，67页PDF精炼

---

## 核心认知（三句话）

1. **内容要素化** — 爆款可以被解构为最小单元
2. **获取专项化** — 分类、聚焦、突破
3. **创作乘法化** — 要素A × 要素B = 裂变内容

---

## 两大内容思维

### 思维1：要素思维

**定义：** 把复杂系统解构为一个个最小单元，再基于最小单元去重构

**要素的特征：**
- 重要 — 容易爆
- 可重复 — 多次爆
- 独立 — 组合爆

**要素可以是：** 一个镜头、词汇、一段话、需求、人群、类型、演员、其他特征

**核心公式：** 爆款内容 = 爆款要素的叉乘（最爆的画面 × 最爆的句子）

### 思维2：专项思维

**定义：** 把庞大的内容系统拆解为一个个独立的内容专项，进而更深度拿到结果

**方法：** 分类 → 聚焦 → 突破

**落地方式：** 拆解自己近期 top10 → 看什么要素是关键的 → 专项突破

## 三大模型

### 模型1：要素连乘模型

百万GMV = S级要素1 × S级要素2 × ... × S级要素N

- 你没法一次性把所有要素打磨到S级
- 但可以把一个个要素分别打磨到S级，再拼到一起

### 模型2：内容要素金字塔模型

14个要素的分类框架（见下文），用于指导拆解和生产

### 模型3：内核-外壳模型

- **内核：** 产品核心卖点、转化逻辑（稳定）
- **外壳：** 前3秒的钩子、吸引方式（可持续测试更换）
- 80%的公司80%的内容类型是混剪，外壳-内核模型非常适用
- 原则：内核稳定的前提下，不断测试外壳；尽量别外壳和内核都同时是新的

## 14个内容要素

### 脚本5个主要素

#### 要素1：需求

- 所有购买行为的本质都是需求被满足
- **需求二分法：**
  - 视觉/脚本二分 — 有些通过视觉激活，有些通过脚本激活
  - 问题/欲望二分 — 问题：不解决不舒服；欲望：想要但不影响现状
- 大需求 = 大消耗（必要不充分条件）
- **验证方法：** 外验（对标品牌是否做过）+ 内验（自己测试）
- **工具：** 需求专项分析表

#### 要素2：人群

- 指**内容人群**，不是投放工具里的8大人群
- 特征：人群可带入、需求可击中
- 新的人群 = 新的增量
- **内容定人群：** 目标人群想看什么内容 → 基于偏好造梦
- **遍历法：** 大标签列举 → 标签细分 → 找到机会人群
- 需求 × 人群可以裂变（十万饼和百万饼的区别）

#### 要素3：类型

- 只有看到生产方式，才有落地落好的可能
- **常见大类型：** 混剪、真人口播、机制、剧情、营销号、新闻号、切片、IP、vlog
- **大类型做方向决策，小类型做精确决策**
- 例：剧情 → 微剧情 / 长剧情 / 搬运剧情（生产方式完全不同）

#### 要素4：特征

- 可独立存在、可排列组合的内容特征
- 例：短句+快切、切割+水果、权威+选择题
- **特征的不同表达，是内容的细化能力**
- 延伸：**小趋势** — 确定性极高的流量饼（周期性流行的元素）

#### 要素5：品类PK

- **思考顺序：**
  1. 我的产品解决哪些需求？
  2. 这些需求过去由什么老品类解决？
  3. 老品类有什么问题？
  4. 我的新品类如何解决了这些老问题？
- **选谁PK：** 够大、够痛、够相关
- PK对象可以是非品牌/非商品（如课本）
- **AABB结构工具：** A品类有问题因为A设计 → B品类设计不同所以没有A的问题（都要有视觉证明）
- **思考方向：** PK过去的自己 / 非商品对标 / 本品类设计缺陷
- 品类PK无对错，有大小，要排优先级，越大越好

#### 要素6：用户痛点

- **分类：** 显性痛点（用户有认识）、隐性痛点（激发后认同）
- 痛点挖掘和痛点表达是两个不同专项
- **与品类PK的区别：** 痛点侧重用户自身的问题，品类PK侧重品类解决方案的问题
- **寻找方法：** 竞品爆量提取、联想词、关键词搜索、评论区高赞、电商评论区、脑暴、AI、产品经理、用户访谈、代入体验

#### 要素7：USP（独特卖点）

- USP = 不同（对用户重要的不同）
- **来源：** 独特工艺技术、独特设计、独特成分、独特理念、独特人群/场景
- **内容结构 1+3+O：** 1个USP + 3个支撑卖点 + 其他
- **品类PK、痛点、USP的关系：**
  1. 发现用户固有痛点（痛点）
  2. 发现品类固有问题（品类PK）
  3. 从中发现差异化价值，作出独特可信赖、可交付的设计（USP）
- 白牌的USP吹牛逼，品牌的USP可交付

#### 要素8：权威

- 可以是免费的（文案、视觉、场景暗示）也可以是付费的（明星）
- 权威是一种思维方式，融会贯通在内容细节中
- **明星要素：** 关联度、情绪、背书、实拍关联性、混剪深度、金句储备、AI、热点
- **明星5大专项：** 选择、实拍、混剪、AI口播、网络素材挖掘
- 80%的公司明星预算至少浪费一半

#### 要素9：场景

- **A-时节场景：** 618/双十一、应季、节假日（相对容易落地和持续拿量）
- **B-使用场景：** 用户有直接使用需求的场景
- 多使用场景的重复 = "一物多用"（常见要素）
- 场景 × 其他要素可以叉乘

### 视觉5个主要素

#### 要素10：同框

- 出现在同一个场景框里的人、场、事、声音会被认为具有相似属性
- 货还是那个货，同框里的人变了、场变了，对货的感觉也变了
- **45度法则：** 目标人群认为"向上45度生活"的样子 — 太高无关，太低无感
- **同框专项：** 演员选择、场景选择、桌拍
- 三类演员：机制类（口条表现力）、效果类（颜值高）、演绎类（微表情演技）

#### 要素11：超级效果

- 一个看了就想买的镜头，呈现产品核心效果，让用户强烈有感
- 不是good是wow，不是还行是卧槽
- **两个方向：** 画面型超级效果（某个画面） / 流程型超级效果（10秒一整套动作）
- 大盘70%-80%的消耗来自超效型品类
- 痛点、效果、品类PK都是需求的来源，只是不同视角：痛点和品类PK是负向（-），超级效果是正向（+）

#### 要素12：符号动作

- 一个或一组可以放大产品效果/卖点的、具有停留+转化效果的动作
- **设计原则 — 合理的反常：** 99%的人这样做，你在合理范围内做不同
- 转化需要合理（圈到合适人群），停留需要不同（吸引注意力）
- **5个小专项：** 模特、动作、道具、实验、穿搭

#### 要素13：实验

- 有说服力的、可视觉化的、可过审的实验，完成卖点证明
- **随手小实验：** 5秒内完成的简单日常化实验
- **强化组合：** 实验+效果、效果+实验+动作
- **描述可视法：** 回看纯描述卖点 → 抽象副词/形容词变为名词/视觉词 → 设计实验镜头 → 停留效果放大

#### 要素14：情绪

- 包括但不限于：愤怒、笃定、质疑、悲伤、欢喜、遗憾、悔悟
- 常作为外壳使用，吸引停留
- 达人向内容情绪要素都很重要（千川和星图）
- 好用的头达/明星 = 强情绪 + 强背书

## 关键工具和方法

| 工具/方法 | 用途 |
|---|---|
| AABB结构 | 品类PK表达框架 |
| 1+3+O结构 | USP内容结构 |
| 需求专项分析表 | 系统盘点需求要素 |
| 遍历法 | 寻找机会人群 |
| 描述可视法 | 将抽象卖点转化为可视实验 |
| 外验+内验 | 验证要素是否有效 |
| 合理反常法 | 设计符号动作 |

## 高层认知

- 内容是一种科学，不是艺术/经验；科学的本质是可复现性
- 内容是一种能力，而非信息差
- 不要用内容的勤奋，来掩盖产品的懒惰
- GMV = 产品分 × 内容分
- 品牌升级路径：搬运 → 挖掘 → 创新
- 内容电商的创新不是天马行空，而是要素提炼后的重新组合
- 不吃内容的苦，就要吃流量的苦、GMV的苦、ROI的苦
